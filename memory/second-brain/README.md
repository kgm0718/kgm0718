# Second Brain - Personal Knowledge Base

## Overview
This is a searchable knowledge base where you can store anything you want to remember.

## How to Use
1. **Save:** Just tell Merry "기억해: [내용]" or "Save this: [content]"
2. **Search:** Ask "Search for: [keyword]" or "Find: [query]"
3. **Browse:** Check this folder for all saved memories

## Categories
- ideas/ - Random ideas and thoughts
- articles/ - Article summaries and notes
- people/ - People you met and info about them
- projects/ - Project notes and ideas
- learning/ - Things you're learning
- miscellaneous/ - Everything else

## File Naming
- YYYY-MM-DD_HH-MM-SS_category_title.md
- Example: 2026-02-18_15-30-00_ideas_startup-app.md
